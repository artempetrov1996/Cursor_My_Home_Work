Команды для Гита:

  • git config --global user.name blabla - Позволяет установить имя пользователя
  • git config --global user.email blabal@bla - Позволяет изменить почту пользователя
  • git add - Позволяет добавить содержимое файла в индекс
  • git init - Позволяет создать скрытую папку которая содержит локальную инфо
  • git status - Позволяет показать статус рабочего дерева и узанать какие файли не были дабавлены
  • git commit -m "....." - Позволяет запись изменения в репозиторий
  • git log - Позволяет показать историю виполненых коммитов
  • git checkout + (название ветки) - Позволяет переходить с ветки на ветку
  • git checkout -b ..... - Позволяет создать новую ветку
  • git branch -M - Позволяет создавать удалять управлять ветками
  • git push - Позволяет отправить изменения в репозиторий
  • git pull - Позволяет перенести всю инфо с сервера
  • git merge - Позволяет объединяет две или более ветки развития вместе
  • git clone Позволяет загрузить копиию репозитория

.gitignore

Нужен для того что бы добавить в него исключения для любого файла что бы он не отправлялся на сервер что бы добавить файл в (.gitignore) нужко добавить имя файла а содержимое (.gitignore)
